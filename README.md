# Variable Phone Holder
A convenient 3d print with M4 Screws and Nuts to hold your phone up in any direction and angle. 
<img width="2220" height="3224" alt="20260623_111249" src="https://github.com/user-attachments/assets/15964d44-6ed3-4c8a-93b4-32f4dfefcec5" />
To Assemble this 3d Print, check the Part Assembly in Onshape, you can view the model without an account and all parts are labeled for easy assembly. 
For People with M3 screws or others, you may remix this project using the step files and create your own variations!
The Onshape Link Is [Here!](https://cad.onshape.com/documents/a7d6c8c886c25331c3d3122e/w/f876ccd8f27584be0431cb56/e/11fe752b5a035021284cbc45?renderMode=0&uiState=6a3ad707757be744bdc774c1)
